# Oracle extra codes

## Navigation

- Show user,

```sql
show user;
```

- List all users,

```sql
select * from all_users;
```

- List all tables,

```sql
select * from all_tables;
```

- List all views,

```sql
select * from all_views;
```

- List all profiles,

```sql
select * from dba_profiles;
```

- List all triggers,

```sql
select * from all_triggers;
```
