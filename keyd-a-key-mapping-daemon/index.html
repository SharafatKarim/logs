Keyd - a key mapping daemon⌗

Installation⌗ 

sudo pacman -S keyd

Systemd service⌗ 

sudo systemctl start keyd

Use sudo systemctl enable keyd to start keyd on boot.

Config⌗

Edit the /etc/keyd/default.conf file like,

[ids]

*

[main]

leftshift = wakeup

To know the key name and other stuff, use keyd -h.
