# Kahoot! - App Review

## URLs

- [Homepage](https://kahoot.com/)
- [Android app](https://play.google.com/store/apps/details?id=no.mobitroll.kahoot.android)
- [Teachers web app](https://create.kahoot.it/)
- [Students web app](https://kahoot.it/)

## Free Features

- 10 participant limit
- Multiple Choice and True/False questions
- Classic Live game mode
- Access to community-created kahoots
- Basic mobile study tools (flashcards)

## Premium Features

- Up to 2,000+ participants
- Advanced questions (Puzzle, Type Answer, Word Cloud, Open-ended, Polls)
- AI Question Generator (from URLs, PDFs, or prompts)
- Slide imports (PowerPoint/Google Slides)
- Detailed analytics and progress reports
- Custom themes and branding
- Team mode and tournament hosting
